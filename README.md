# NAIB Mobile Banking Solution (Prototype)

A comprehensive mobile banking prototype developed during my internship at the **North African International Bank (NAIB)**. This project consists of two distinct applications designed to modernize client interactions and administrative workflows.

## 🚀 Project Overview
This solution was built to demonstrate a secure and user-friendly mobile banking experience. It bridges the gap between client self-service and administrative oversight.

### 📱 Client Application
* **Secure Authentication:** Login system for verified users.
* **Account Management:** Real-time view of account balances and recent transactions.
* **Service Requests:** Interactive modules to request checkbooks or track debt status.

### 🛠️ Admin Application
* **User Management:** Tools to manage company and individual user accounts.
* **Operational Oversight:** Review financial data and process pending client requests.
* **Data Control:** Centralized dashboard for banking operations.

## 🛠️ Tech Stack
* **IDE:** WinDev Mobile
* **Language:** WLanguage
* **Database:** HFSQL (Mobile-optimized database management)
* **Design:** Custom UI/UX focused on banking security and clarity.

## 📂 Project Structure
* `/Source`: Contains the WinDev Mobile project files and WLanguage logic.
* `/Documentation`: Includes the full Internship Report and the Prototype Presentation.

All data shown in the screenshots and the prototype (names, emails, account numbers) consists of **mock data** used for demonstration purposes. No sensitive or real banking information from NAIB was used in this repository.
